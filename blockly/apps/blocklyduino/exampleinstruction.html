Changing the maximum length of the sequence

The maximum sequence length is a variable in the code which takes an integer value (a whole number). This variable is used in the code to determine the maximum length that the sequence can have. The sequence is the order in which the LED lights light up. By changing the value of this variable, the sequence has the possibility to be longer or shorter. If you increase the maximum sequence length, the sequence can be longer. If you decrease the maximum sequence length, the sequence can be shorter.
